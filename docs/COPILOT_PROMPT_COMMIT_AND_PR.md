# Copilot Prompt — Commit + PR Text (Copy/Paste)

Du bist mein Git-Coach. Schaue dir `git diff --staged` an und gib mir:

1) Commit Summary im Format: `type(scope): message`
   Types: feat, fix, docs, chore, refactor
   Scopes: portal, make, billing, security, docs, ci

2) Commit Description (2–4 Zeilen), inkl. optional genau einer Zeile:
   `Closes #<issue>` oder `Fixes #<issue>` (wenn passend)

3) PR Title + PR Description (kurz), inkl. derselben `Closes #<issue>` Zeile.

Antwortformat:
- Commit Summary:
- Commit Description:
- PR Title:
- PR Description:
